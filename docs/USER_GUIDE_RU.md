# Hermes — Руководство пользователя

> **Hermes** — десктоп-приложение для автоматической торговли на Forex и
> крипторынках. Разработано BAI Core ([baicore.kz](https://baicore.kz)).

---

## 1. Установка

### Windows
1. Скачайте `Hermes-Setup-1.0.0.exe` с [hermes.baicore.kz](https://hermes.baicore.kz).
2. Дважды кликните по файлу.
3. Пройдите wizard на русском (5 экранов: приветствие → лицензия → папка → ярлыки → готово).
4. Готово — Hermes запустился в собственном окне.

#### 🛡 Если SmartScreen ругается «Защитник Windows предотвратил запуск…»
Это **нормально для новых программ без купленного сертификата**. Файл не вирус — это
запасная мера от Microsoft, пока приложение не наберёт репутацию (обычно несколько сотен
установок). Чтобы запустить:

1. В синем окне нажмите ссылку **«Подробнее»** (мелким шрифтом).
2. Появится кнопка **«Выполнить в любом случае»** — нажмите её.

Альтернатива — проверьте контрольную сумму перед запуском, чтобы убедиться что файл
не подменили:
```powershell
Get-FileHash Hermes-Setup-1.0.0.exe -Algorithm SHA256
```
Сравните с `SHA256SUMS.txt`, который лежит рядом со ссылкой на скачивание.

### macOS
1. Скачайте `Hermes-1.0.0.pkg`.
2. Дважды кликните → пройдите установщик.
3. Запустите Hermes из Launchpad.

#### 🍏 Если macOS говорит «Невозможно открыть, разработчик не идентифицирован»
Самый быстрый путь — двойным кликом запустить файл `unblock-hermes.command`,
который лежит рядом со ссылкой на скачивание. Он снимет quarantine attribute и
Hermes запустится при следующем клике.

Ручной путь:
1. **System Settings → Privacy & Security**.
2. Прокрутите вниз — увидите строку «"Hermes" was blocked from use because it is not from an identified developer».
3. Нажмите **«Open Anyway»**.
4. При следующем запуске Hermes — нажмите **«Open»** в диалоге.

После одного раза macOS запоминает разрешение, и больше предупреждений не будет.

Проверка контрольной суммы:
```bash
shasum -a 256 Hermes-1.0.0.pkg
```
Сравните с `SHA256SUMS-macos.txt`.

### Linux
В разработке. Следите за обновлениями на baicore.kz.

## 2. Первый запуск

После запуска Hermes покажет splash, а затем — Onboarding (3 экрана):

### Экран 1. Создание мастер-пароля
- Минимум 6 символов. **Запишите в надёжное место** — восстановить нельзя.
- Все API-ключи бирж шифруются этим паролем (Argon2id + Fernet AES-128).

### Экран 2. Подключение брокера
Выберите тип счёта:

**MT5 (Forex)** — Hermes автоматически найдёт серверы установленных терминалов:
- Сервер (например, `Exness-MT5Trial`)
- Логин (номер счёта)
- Инвестор-пароль

Нажмите «Проверить» — Hermes пробно подключится и покажет баланс. Если ОК — «Сохранить и далее».

**Криптобиржа (Binance / Bybit / OKX)**:
- API Key
- API Secret
- (для OKX) API Passphrase
- Включите «testnet» если хотите попрактиковаться без реальных средств.

### Экран 3. Стратегия
Выберите пресет:
| Пресет | Риск | Описание |
|---|---|---|
| 🛡 **Консервативный** | низкий | Малый риск, ~3-5 сделок в неделю |
| ⚖ **Сбалансированный** | средний | Оптимальный баланс, ~7-12 сделок/нед |
| ⚔ **Агрессивный** | высокий | Частые входы, больший размер ордеров |
| ✨ **Auto** | адаптивный | Hermes сам подстраивается под рынок (рекомендуется) |

Нажмите «Запустить Hermes» — попадёте на главный экран, бот начнёт работу.

## 3. Главный экран («Олимп»)

Сверху — статус бота, рыночный режим (Тренд / Флэт / Высокая волатильность), кнопки управления.

**KPI карточки**:
- **Equity** — текущее состояние счёта.
- **Сегодня** — P&L за период.
- **Просадка** — насколько упали от пика; стрелка справа показывает зоны (зелёная — безопасно, бронза — внимание, красная — близко к hard-stop).
- **Win-rate** — процент прибыльных сделок.

**Per-pair regime row** — Hermes классифицирует каждую пару:
- 📈 Тренд — направленное движение, работает гридовая стратегия.
- ⚖ Флэт — боковик, используются меньшие отступы и больший TP.
- ⚡ Волатильность — сильные движения, бот приостанавливает новые входы.

**Кривая equity** — обновляется в реальном времени через WebSocket.

**Открытые позиции** — таблица с возможностью закрыть позицию по одной (крестик справа).

**Аварийная остановка** — большая красная кнопка, закрывает **все** позиции немедленно. Двухступенчатое подтверждение защищает от случайного нажатия.

## 4. Стратегия (раздел в меню)

Можно выбрать пресет или вручную настроить параметры:
- **Расстояние между ордерами** (пипс)
- **Множитель лота** — насколько растёт позиция на каждом уровне
- **Базовый лот** — стартовый размер
- **Уровней сетки** — максимум усреднений
- **Take Profit (%)** — цель по equity
- **Stop drawdown (%)** — уровень остановки новых ордеров
- **Max просадка портфеля (%)** — hard stop, всё закрывается

Hermes валидирует параметры в реальном времени:
- 🔴 **Ошибка** — например, лот × множитель × уровни потребует > 50% депозита.
- 🟡 **Предупреждение** — между 25% и 50% депозита; возможен margin call при сильном движении.

## 5. Бэктест

`Меню → Бэктест` → выберите период (30/60/90/180/365 дней) → «Запустить».

Hermes пройдёт текущей стратегией по истории биржи и покажет:
- **Sharpe Ratio** — отношение доходности к риску.
- **Доходность** — итоговый прирост.
- **Max DD** — максимальная просадка.
- **Profit factor** — отношение прибылей к убыткам.

Каждый прогон сохраняется в историю — можно сравнить результаты разных версий стратегии.

---

## 6. Оптимизация

`Меню → Оптимизация` → выберите trials (30/60/100/200) и период → «Запустить».

Optuna перебирает параметры стратегии и в реальном времени показывает таблицу top-25 проб
по Sharpe. После завершения нажмите **«Применить лучшие параметры»** — активная стратегия
обновится. Старая версия сохранена через `parent_id` для отката.

## 7. Адаптивная торговля (Auto-режим)

Если выбран пресет **Auto**, каждое **воскресенье в 03:00 локального времени** Hermes
запустит автоматическую калибровку:

1. Загружает 90 дней рынка.
2. Walk-forward split 70/30.
3. Optuna 100 trials на in-sample.
4. Валидация на out-of-sample.
5. **Champion-challenger**: новые параметры применяются только если они лучше старых на ≥5% по Sharpe **И** просадка не выросла больше чем на 20%.
6. Если калибровка проиграла — старые параметры остаются нетронутыми.

Историю калибровок и кнопку отката найдёте в разделе «Стратегия».

## 8. Управление с телефона

`Меню → На телефон` → нажмите **«Открыть туннель»**.

Hermes поднимет защищённый ngrok-канал и покажет:
- **URL** — адрес туннеля (можно скопировать).
- **PIN** — 6-значный код для доступа (меняется автоматически раз в 24 часа).
- **QR-код** — отсканируйте с телефона, откроется PWA-интерфейс.

> **Безопасность**: 5 неудачных попыток PIN → туннель блокируется на 10 минут.
> Когда мобильный доступ не нужен — закройте туннель кнопкой «Закрыть».

На iPhone / Android в Safari / Chrome нажмите **«Поделиться → Добавить на главный экран»** —
Hermes установится как нативное приложение.

## 9. Уведомления

`Меню → Уведомления` — Hermes отправляет события (открытие/закрытие сделки, ошибки) на:

### Web Push
Кнопка «Включить» → разрешите уведомления → Hermes будет слать push прямо в браузер
(или на телефон, если установили PWA).

### Telegram
1. Создайте бота через [@BotFather](https://t.me/BotFather): `/newbot` → имя → username → получите **token**.
2. Отправьте боту любое сообщение, затем напишите [@userinfobot](https://t.me/userinfobot) — получите свой **chat_id**.
3. В Hermes введите token и chat_id → «Подключить». Hermes отправит тестовое сообщение для проверки.

Кнопка **«Отправить тест»** на странице — проверяет все каналы одновременно.

## 10. Безопасность

- 🔐 **Шифрование**: Argon2id (memory=64MB, 3 passes, 4 lanes) → 32-byte ключ → Fernet.
- 📁 **Хранение**: `%APPDATA%\Hermes\credentials.enc` (Windows), `~/Library/Application Support/Hermes/credentials.enc` (macOS).
- 🔒 **3 неудачные попытки** разблокировки → блокировка на 10 минут.
- 🌐 **127.0.0.1 only** — backend слушает только loopback. Удалённый доступ только через осознанно открытый туннель + PIN.
- 🛡 **Валидация** — нельзя сохранить параметры, которые могут привести к margin call.

## 11. Что делать если…

| Ситуация | Что делать |
|---|---|
| Забыли мастер-пароль | Нет восстановления. Удалите `credentials.enc` из data dir и создайте заново — потеряете подключения, но настройки стратегии останутся. |
| Бот не открывает сделки | Проверьте: брокер активен? Стратегия активна? Рынок в режиме `high_vol`? Логи на `Меню → Журнал`. |
| MT5 ошибка «не удалось войти» | Включите Algo Trading в самом MT5 (Tools → Options → Expert Advisors → Allow algorithmic trading). |
| Туннель не открывается | По умолчанию ngrok работает анонимно. Для лучшей стабильности задайте `BCT_NGROK_AUTHTOKEN` через Settings → Environment. |
