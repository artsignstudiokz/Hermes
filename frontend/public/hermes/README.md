# Hermes imagery

Place public-domain photos of Hermes statues here. Recommended sources:

- **Hermes of Praxiteles** (Olympia Archaeological Museum) — Wikimedia Commons
- **Hermes Belvedere** (Vatican Museums) — Wikimedia Commons
- **Hermes Logios** (Louvre) — Wikimedia Commons

Suggested filenames (referenced from the app):

- `hermes-praxiteles.jpg` — used on Onboarding hero
- `hermes-belvedere.jpg` — used on Unlock screen background
- `caduceus-relief.jpg` — used on Settings/About header
- `marble-texture.jpg` — full-page background texture (optional)

All images should be JPEG, max 2000px wide, < 400KB each. Save to this
folder and they will be picked up automatically by the components that
reference `/hermes/<filename>`.
